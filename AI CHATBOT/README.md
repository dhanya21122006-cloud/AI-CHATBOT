 🤖 AI Chatbot Project

 📌 Overview

This project is a simple AI-powered chatbot built using Python and Streamlit. It provides an interactive chat interface where users can communicate with the bot and receive intelligent responses based on input keywords. The chatbot can also analyze uploaded PDF documents and respond based on their content.

---

 🚀 Features

* 💬 Interactive chat interface (like ChatGPT)
* 🧠 Keyword-based intelligent responses
* 📄 PDF upload and basic content analysis
* ⚡ Instant responses with smooth UI
* 🔒 Works offline (no API required)

---

 🛠️ Technologies Used

* Python
* Streamlit
* PyPDF2

---

 🧠 Prompt Design

The chatbot uses a rule-based prompt design approach. It analyzes user input and matches keywords to generate appropriate responses.

 Examples:

* "hello" → Greeting response
* "python" → Information about Python
* "project" → Explanation of the chatbot project
* PDF keywords → Matches words from uploaded document

---

 ⚠️ Limitations

* Not a real Large Language Model (LLM)
* Limited understanding of complex sentences
* Responses are based on predefined rules
* PDF analysis is keyword-based only

---

 💡 Sample Outputs

**User:** Tell me about Python
**Bot:** Python is a powerful programming language used for AI, web development, and data science.

**User:** Hello
**Bot:** Hey there! 😊 I'm your AI chatbot.
